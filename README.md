# Projeto: Vinharia-Agnello

Integrantes do grupo: Allan Ragazzo Freire, Leonardo Scalisse Silva, Pietro Schimidt Routolo, Danilo Roque Souza Machado e Valdemar da Rocha Formiga neto 
Turma: 1ESPJ-26

Desenvolvimento de um sistema de gerenciamento de vinhos, com um sistema de cadastro básico e uma página index.html com códigos em JavaScript.
O código javascript foi implementado com o sistema de cadastro manual de vinhos, onde será solicitado ao usuário o nome, o tipo, o ano de safra, a quantidade em estoque e o país de origem do vinho. O cadastro irá continuar se repetindo até quadno o usuário desejar, escolhendo "ok" no botão que irá surgir no rodapé da página.
